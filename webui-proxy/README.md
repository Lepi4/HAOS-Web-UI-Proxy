# Web UI Proxy (Home Assistant Add-on)

Проксирует локальные веб-интерфейсы через Ingress Home Assistant.

## Настройка

В конфигурации аддона задайте список устройств с названиями:

```yaml
targets:
  - name: "Proxmox"
    url: "192.168.1.30:8006"
  - name: "MQTT Dashboard"
    url: "192.168.1.70:18083"
```

Или используйте простой формат (без имён):

```yaml
targets:
  - 192.168.1.10
  - 192.168.1.11:8080
```

Откройте аддон через Ingress и выберите нужный интерфейс в списке.
